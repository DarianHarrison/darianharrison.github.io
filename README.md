---

To test in a local environment
```bash
git clone https://github.com/DarianHarrison/darianharrison.github.io/
cd darianharrison.github.io
git submodule update --init --recursive
zola serve
```

Page is set up to automatically update upon push
```bash
git add -A ; git commit -m "2026-02-20" ; git push
```
